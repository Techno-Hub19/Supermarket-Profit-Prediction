# Supermarket Profit Prediction

## Overview
This project uses machine learning to predict supermarket profits based on historical sales data and key business metrics. The model analyzes factors like product sales, advertising spend, and market conditions to forecast future profitability.

## Key Features
- **Data Analysis**: Exploratory analysis of supermarket sales patterns
- **Predictive Model**: ML algorithms to predict profit margins
- **Easy to Use**: Simple implementation for quick profit forecasting

## Getting Started
This project provides tools to help supermarkets optimize their operations and make data-driven decisions to increase profitability.

## Prediction Output

The model takes two key inputs:
- **Sales**: The revenue amount from the order
- **Discount**: The percentage discount offered on the order

## Predictions & Conclusions

Based on the input values, the model provides:

1. **Binary Classification**: 
   - **Profitable** - The order is likely to generate profit
   - **Not Profitable** - The order may result in loss or minimal profit

2. **Probability Scores**: 
   - Confidence levels showing how likely each outcome is
   - Helps understand the model's certainty in its prediction

### Key Insights:
- Orders with high sales and low discounts tend to be more profitable
- Heavy discounts significantly impact profitability
- This helps businesses make informed decisions on pricing and discount strategies
